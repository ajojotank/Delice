API Documentation

This document provides information on how to use the API endpoints with example cURL commands.

User Endpoints

Login User

Logs in a user by checking username and password hash.

POST /login

Example cURL command:

curl -X POST -d "username=chefjoash&password=hash1" https://lamp.ms.wits.ac.za/home/s2670867/login.php

Register User

Registers a new user with username, name, and password.

POST /register

Example cURL command:

curl -X POST -d "username=newuser&name=New User&password=newpassword" https://lamp.ms.wits.ac.za/home/s2670867/register.php

Recipe Endpoints

Get All Recipes

Retrieves all recipes with their details.

GET /recipes

Example cURL command:

curl -X GET https://lamp.ms.wits.ac.za/home/s2670867/get_recipes.php

Get Recipe by ID

Retrieves a single recipe by its ID.

GET /recipes/{id}

Example cURL command:

curl -X GET https://lamp.ms.wits.ac.za/home/s2670867/get_recipe_by_id.php?id=1

Add New Recipe

Adds a new recipe to the database.

POST /recipes

Example cURL command:

curl -X POST -F "title=New Recipe" -F "description=Delicious new recipe" -F "author_id=1" -F "imageFile=@/path/to/image.jpg" https://lamp.ms.wits.ac.za/home/s2670867/add_recipe.php

Meal Planning Endpoints

Get Meals for User

Retrieves all meals planned for a specific user.

GET /meals/{user_id}

Example cURL command:

curl -X GET https://lamp.ms.wits.ac.za/home/s2670867/get_meals_for_user.php?user_id=1

Add Meal to Planner

Adds a meal to a user's meal planner.

POST /meals

Example cURL command:

curl -X POST -d "user_id=1&recipe_id=1&day=Mon&meal_type=Lunch" https://lamp.ms.wits.ac.za/home/s2670867/add_meal.php

Delete Meal

Deletes a meal from the meal planner.

DELETE /meals/{id}

Example cURL command:

curl -X DELETE https://lamp.ms.wits.ac.za/home/s2670867/delete_meal.php?id=1

Allows searching recipes by title or ingredients.

Example cURL command:
